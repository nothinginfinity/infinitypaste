# ∞ InfinityPaste

> **Copy many. Paste once.**

A clipboard queue PWA for iPhone. Save multiple items from anywhere,
then paste them one by one — or all at once — wherever you need them.

**Live app:** [https://nothinginfinity.github.io/infinitypaste](https://nothinginfinity.github.io/infinitypaste)

---

## The Problem

iOS only holds one thing in the clipboard at a time. If you're working
with an AI assistant, copying multiple code blocks, notes, or prompts
across sessions means constant back-and-forth switching.

InfinityPaste lets you queue up everything first, then paste it all
when you're ready.

## How to Use

1. Open the app → **Collect tab**
2. Paste or type content → tap **+ Add to Queue**
3. Repeat for every item you want
4. Switch to **Queue tab** → tap **Copy** on any item, or **Copy All**
5. Paste wherever you need it

## Install to iPhone Home Screen

1. Open [https://nothinginfinity.github.io/infinitypaste](https://nothinginfinity.github.io/infinitypaste) in Safari
2. Tap the Share button (□↑)
3. Tap **Add to Home Screen**
4. Tap **Add**

Works offline after first load.

## Features

- 📋 Queue unlimited clipboard items
- 🏷️ Optional labels per item
- 🔢 Numbered items for easy reference
- 📦 Copy All — combine everything with separators
- ⚙️ Auto-clear after Copy All (optional)
- 💾 Persists across sessions (localStorage)
- 📴 Works fully offline (service worker)
- 🌑 Dark mode only, iPhone-native feel

## Stack

Vanilla JS + CSS. Zero dependencies. No server. No account required.
